# my-games

スマホ向け1画面完結の **2D 4対4コマンドバトル** の最小実装です。

## 最小仕様
- 味方4体 vs 敵4体の固定バトル。
- 行動順は素早さ順。
- 味方ターンは3スキルから選択（CT表示あり）。
- 単体対象スキルはユニットをタップして対象選択。
- 敵ターンは簡易AIが自動行動。
- 状態効果は「攻撃UP / 防御DOWN / 気絶 / 継続回復」のみ。
- 敵全滅で勝利、味方全滅で敗北。

## 画面構成
- 左: 味方4体（名前・HP・状態表示・行動中ハイライト）
- 右: 敵4体（同上）
- 中央: 行動ログと現在ターン情報
- 下: 現在行動中の味方のスキル1〜3ボタン

## 実行方法
`index.html` をブラウザで開くだけで動作します。
